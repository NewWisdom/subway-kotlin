## subway-kotlin
죽이되든 밥이되든... 일단 해보는 꼬뜰린과 jpa로 쟈철 미션 구현 🦦

* 몰라 일단 부딪혀보면서 배우는 jpa
* service-apply 참고하면서 구현해보기
* 꼬틀린 트러블 슈팅 기록 기록
* 각각 미션에서 아쉬웠던 부분 보완하기
* 갹각 미션에서 욕심내고 싶은 부분 도전하기

### 시도한 부분
* jpa 사용
* 코틀린 dls로 rest-assured 테스트 구현
* 테스트 코드 가독성 신경쓰기
